Listas OL e UL

Para criar uma lista, abrimos ela com a tag "ol" ou "ul", ol serve para uma lista de numeros, numeros romanos ou letras, e a ul para pontos, circulos e quadrados. Dentro dessas duas tags podemos colocar o "type" para escolher o formato e "start" de onde voce que iniciar a lista. E para consigir ordenar a lista se usa a tag "li"

EXEMPLO de ol

  1. Acordar
  2. Ligar para o João

EXEMPLO de ul

  • Pão
  • Leite
